## ğŸ“¦ Installation & Setup

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side dependencies required
- Recommended: VS Code or similar code editor

### Step 1: Clone the Repository
```bash
git clone https://github.com/padmapriya-28/My-Portfolio.git
cd My-Portfolio
```

### Step 2: Open in Browser
Simply open the `index.html` file in your web browser:
```bash
# On Windows
start index.html

# On macOS
open index.html

# On Linux
xdg-open index.html
```

### Step 3: Local Development Server (Optional)
For development purposes, use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if you have http-server installed)
http-server
```

Then navigate to `http://localhost:8000` in your browser.

The portfolio is fully responsive and optimized for all devices:

| Device | Breakpoint | Optimizations |
